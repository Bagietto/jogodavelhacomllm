# ADR-0002 - Integracao LLM Direta no Angular

- Status: Aceito
- Data: 2026-03-12

## Contexto
Projeto de portfolio academico com foco em simplicidade de entrega.

## Decisao
Realizar chamadas diretas do frontend Angular 21.x para OpenRouter.

## Consequencias
- Pro: menor complexidade de infraestrutura.
- Contra: exposicao de chave no cliente (risco aceito para MVP).
