# ADR-0003 - UI 2D First no MVP

- Status: Aceito
- Data: 2026-03-12

## Contexto
Projeto de portfolio com foco em entrega rapida e jogabilidade.

## Decisao
Implementar tabuleiro 2D no MVP.

## Consequencias
- Pro: menor complexidade e tempo de desenvolvimento.
- Pro: facilita foco em IA, regras e UX.
- Contra: tabuleiro 3D fica para evolucao futura.
