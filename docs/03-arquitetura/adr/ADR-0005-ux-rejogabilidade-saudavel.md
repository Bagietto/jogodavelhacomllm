# ADR-0005 - UX de Rejogabilidade Saudavel

- Status: Aceito
- Data: 2026-03-12

## Contexto
Desejo de aumentar engajamento do jogo sem estimular comportamento compulsivo.

## Decisao
- Adotar UI atraente com feedback claro de jogo.
- Proibir dark patterns de retencao compulsiva.
- Exibir CTA neutro de continuidade ao fim da partida.

## Consequencias
- Pro: melhor experiencia e etica de produto.
- Pro: melhor apresentacao de portfolio.
- Contra: exige validacao de UX adicional.
