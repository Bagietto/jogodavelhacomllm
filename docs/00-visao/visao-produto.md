# Visao do Produto

## Produto
Jogo da Velha com IA (LLM) para web.

## Problema
Permitir jogar quando quiser sem depender de outro humano.

## Publico-Alvo
Geral.

## Objetivo de Negocio
Projeto academico para portfolio no GitHub.

## Escopo MVP
- Partida humano vs IA.
- Tabuleiro 2D.
- Placar acumulado: Humano x IA x Empates.
- Interface visual atraente com foco em rejogabilidade saudavel.
- Escolha de quem inicia.
- Escolha de simbolo (X ou O).
- Dificuldade: facil, medio, mestre.
- Chat da IA amigavel/engracado.
- Nome fixo da IA na interface (padrao: VelhAIra).
- Idioma da IA: pt-BR, es, en-US.
- LLM conversacional via OpenRouter com modelo padrao e fallback.

## Fora de Escopo
- Jogo online entre humanos.
- Tabuleiro 3D no MVP.

## Criterio de Sucesso
- Jogabilidade fluida sem travamento perceptivel na jogada da IA.
- Media >= 3 partidas por sessao sem uso de padroes viciantes.
